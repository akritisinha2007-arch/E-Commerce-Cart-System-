# E-Commerce-Cart-System-
Python OOPs Project-1
A simple Shopping Cart System built using Python and Object-Oriented Programming (OOP) concepts.

## Files

- `product.py` - Product class
- `cart_item.py` - CartItem class
- `order.py` - Order class
- `main.py` - menu loop and all the logic, run this file

## Features

- Show available products
- Add products to cart
- View cart with total price
- Checkout and place order
- View all orders
- Cancel an order (with refund)
- View wallet balance

## Technologies Used

- Python
- Object-Oriented Programming (OOP)

## Menu Options

1. Show Products
2. Add to Cart
3. Show Cart
4. Checkout
5. Show Orders
6. Cancel Order
7. Show Wallet Balance
8. Exit
